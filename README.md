# 🌱 EcoCiclo

O **EcoCiclo** é um sistema de monitoramento inteligente para coleta seletiva de resíduos recicláveis. O projeto combina **tecnologia web** com **sensores conectados a microcontroladores (Arduino)** para acompanhar o nível de preenchimento de recipientes de coleta e melhorar a eficiência da gestão de resíduos.

A proposta do EcoCiclo é contribuir para cidades mais sustentáveis, facilitando o descarte correto de materiais recicláveis e otimizando a logística de coleta.

---

# 📌 Objetivo do Projeto

O objetivo do EcoCiclo é desenvolver uma plataforma capaz de:

- Monitorar o nível de preenchimento de lixeiras de coleta seletiva
- Enviar alertas quando um recipiente estiver próximo da capacidade máxima
- Disponibilizar um **mapa interativo com pontos de coleta**
- Permitir que administradores acompanhem dados e relatórios de uso
- Incentivar práticas de descarte consciente

Com isso, o sistema ajuda a **reduzir desperdícios, melhorar rotas de coleta e aumentar a eficiência da reciclagem**.

---

# ⚙️ Como o Sistema Funciona

O EcoCiclo é composto por três partes principais:

## 1️⃣ Sensores e Arduino
Sensores instalados nos recipientes de coleta monitoram o nível de preenchimento e enviam essas informações para o sistema.

## 2️⃣ Plataforma Web
O site permite que usuários e administradores acompanhem o funcionamento do sistema.

Entre as funcionalidades disponíveis:

- Visualização de **ecopontos em um mapa interativo**
- **Dashboard com dados e gráficos de monitoramento**
- **Alertas quando lixeiras atingem determinado nível**
- **Cadastro de novos pontos de coleta**
- **Download de relatórios históricos**

## 3️⃣ Painel Administrativo
O administrador pode:

- Monitorar lixeiras que atingiram o limite
- Gerenciar ecopontos cadastrados
- Visualizar dados históricos
- Baixar relatórios de monitoramento
- Acompanhar o uso dos pontos de coleta
